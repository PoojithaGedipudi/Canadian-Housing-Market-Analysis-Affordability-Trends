# Canadian-Housing-Market-Analysis-Affordability-Trends
Power BI dashboard analyzing housing trends and affordability in Canada (2020–2025)

# 🔍 Project Overview

+ Housing affordability has become a major concern in Canada, especially in provinces like Ontario and British Columbia. This project analyzes housing data from 2020 to 2025 to identify trends, regional differences, and key factors affecting property prices.
- The dashboards allow users to interactively explore housing data by province, city, property type, and year.

# 🛠 Tools & Technologies
- Power BI (Dashboard Development)
- Python (Data Cleaning & EDA)
- SQL (Data Queries)
- DAX (Calculated Measures)
- Power Query (Data Transformation)

# 📂 Data Sources
- Canadian Real Estate Association (CREA) – MLS Home Price Index
- Canada Mortgage and Housing Corporation (CMHC)
- Kaggle – Canada Housing Dataset

# 📈 Key Features
- Interactive filters (province, city, housing type, year)
- Forecasting of benchmark housing prices
- Geospatial maps for regional comparison
- Scatter plots to analyze price vs square footage
- KPI cards for quick insights

# 📊 Key Insights
- Housing prices increased significantly between 2020 and 2025 (~48% growth)
- Ontario experienced a price peak in 2022 followed by a market correction
- Square footage shows a strong positive correlation with price
- Apartments remain the most affordable housing option across regions
- Price growth varies significantly by province and housing type
